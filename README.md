Expense Tracker

A minimal Flutter expense tracker implementing:

- Add expenses (amount, category, date)
- Local persistence using sqflite
- Charts (fl_chart)
- Export CSV and share
- Save user name and monthly budget with SharedPreferences

Run:

1. cd to project
2. flutter pub get
3. flutter run

Notes:
- This is a scaffolded app. Ensure Flutter SDK is installed.
- Dependencies are in `pubspec.yaml`.
